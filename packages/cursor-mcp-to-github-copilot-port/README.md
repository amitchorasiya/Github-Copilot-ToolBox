# cursor-mcp-to-github-copilot-port

The **MCP port CLI** is maintained as its own package. Install from npm or clone the tool repo; this folder is a **placeholder** so the monorepo layout stays clear for GitHub.

- **npm:** `npx cursor-mcp-to-github-copilot-port`
- **Related repo:** [Github-Copilot-ToolBox](https://github.com/amitchorasiya/Github-Copilot-ToolBox) — monorepo for the VS Code extension, **IntelliJ** hub plugin (preview), and bundled bridge CLIs. CLI source may live under `packages/` if you vendor it here.
